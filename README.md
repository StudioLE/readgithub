# readgithub
A jQuery plugin to load files from GitHub

## Installation

Script
```
<script src="//cdn.rawgit.com/StudioLE/readgithub/master/readGitHub.js"></script>
```

JS
```
$('[rel~="github"]').readGitHub();
```

HTML
```
<div rel="github" data-repo="username/repository"></div>
```
